# 🌐 Personal Portfolio Website

A modern, responsive, and interactive personal portfolio website built using **HTML, CSS, and JavaScript**.
This project showcases my skills, projects, and experience as a frontend developer.

## 📌 Features

* 🌙 Dark/Light mode toggle
* 🎯 Project filtering system
* ✨ Smooth scroll animations

* 📱 Fully responsive design

* 💬 Contact form (Formspree integrated)
